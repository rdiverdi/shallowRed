Our Project

We are making a chess AI, beginning with minmax searching and alpha-beta pruning, then incorporating neural networks and evolutionary algorithms to optimize the performance.

Preparation and Framing

Background and Context

Chess AIs generally start with a min-max search algorithm which looks ahead a certain number of moves and evaluates that move based on the maximum possible loss due to that move. The algorithm then chooses the move with the smallest maximum loss. Loss is measured by giving a value to each piece, which is then gained or lost when that piece is captured or lost. Alpha-beta pruning is a meathod for speeding up the min-max algorithm by ignoring branches of the seach tree which are obviously worse than already checked branches.
